# Le Voyage d'Orion - Contexte du Jeu

## Description
"Le Voyage d'Orion" est un jeu de société de type "Escape Game" ou jeu d'enquête narratif, qui semble avoir été conçu sur mesure (possiblement pour une demande en mariage, vu la fin). Le joueur explore une carte du ciel, manipule des objets physiques (sextant, gabarits) et résout des énigmes pour progresser dans une histoire.

## Mécaniques de Jeu

### Les Éléments
Le jeu se compose de plusieurs types d'éléments :
*   **Cartes Encyclopédie (Code EN)** : Fournissent des informations contextuelles, des indices sur les constellations ou les objets (ex: "Le Sextant", "Constellation d'Orion").
*   **Cartes Énigme (Code E)** : Posent les problèmes à résoudre. Elles contiennent souvent des citations ou des indices visuels.
*   **Cartes Histoire (Code H)** : Font avancer la narration. Elles sont débloquées en trouvant la bonne réponse (souvent un nombre) à une énigme.
*   **Objets Physiques** :
    *   **Carte du Ciel** : Le plateau principal de recherche.
    *   **Sextant** : Outil de navigation en bois pour s'orienter sur la carte.
    *   **Gabarits** : Formes en bois représentant des constellations (Lapin, Lyre, Sagittaire) avec un Recto et un Verso.
    *   **Autres** : Registre, diapositives, filtre coloré, etc.

### Boucle de Jeu
1.  Le joueur commence avec une **Carte Énigme** et potentiellement des **Cartes Histoire** ou **Encyclopédie** initiales.
2.  Il doit utiliser les indices (textes, visuels) et les **Objets Physiques** (aligner le sextant, poser un gabarit sur la carte du ciel) pour trouver une solution.
3.  La solution est généralement un **nombre** (somme d'étoiles, résultat d'un calcul).
4.  Ce nombre correspond à l'identifiant d'une **Carte Histoire** (ex: Carte 155).
5.  Le joueur pioche cette carte, qui révèle la suite de l'histoire et donne les indices pour l'énigme suivante.

## Résumé Global des Énigmes

Le jeu comporte 16 énigmes (de 0 à 15). Voici leur progression :

*   **Énigme 0 : Le Départ**
    *   *Objectif* : Comprendre le fonctionnement du sextant.
    *   *Action* : Aligner le sextant sur la "Fausse étoile 13" et sommer les numéros pointés.
    *   *Résultat* : Carte Histoire **155** ("L'Ancrage").

*   **Énigme 1 : Le Lapin**
    *   *Objectif* : Identifier la constellation du Lapin.
    *   *Action* : Utiliser le gabarit "Lapin" sur la carte du ciel (espace 110) et sommer les étoiles.
    *   *Résultat* : Carte Histoire **258** ("Rencontre avec le lapin").

*   **Énigme 2 : L'Ombre du Lapin**
    *   *Objectif* : Trouver l'arc d'Orion.
    *   *Action* : Utiliser le verso du gabarit Lapin pour identifier une partie de l'arc d'Orion et sommer les étoiles.
    *   *Résultat* : Carte Histoire **253** ("Le silence se tend").

*   **Énigme 3 : La Flèche**
    *   *Objectif* : Compléter la constellation de Cupidon et Orion.
    *   *Action* : Superposer le gabarit Lapin sur le sextant, aligner les cœurs, tirer des flèches et calculer un "trou noir".
    *   *Résultat* : Carte Histoire **41** ("Libération").

*   **Énigme 4 : La Lyre**
    *   *Objectif* : Trouver la constellation de la Lyre.
    *   *Action* : Utiliser le gabarit "Lyre" recto sur l'espace 70 et sommer les étoiles.
    *   *Résultat* : Carte Histoire **198** ("Harmonie dispersée").

*   **Énigme 5 : Les Cordes**
    *   *Objectif* : Suivre les cordes de la Lyre.
    *   *Action* : Prolonger les traits du gabarit selon le "poids" des cordes pour trouver des étoiles cibles.
    *   *Résultat* : Carte Histoire **285** ("Les cordes retrouvées").

*   **Énigme 6 : L'Accord**
    *   *Objectif* : Trouver un accord de résonance spécifique (197).
    *   *Action* : Sélectionner 3 cordes dont la somme des poids fait 197 et sommer leurs étoiles.
    *   *Résultat* : Carte Histoire **103** ("L'éveil des cordes").

*   **Énigme 7 : La Partition**
    *   *Objectif* : Décrypter une mélodie.
    *   *Action* : Traduire des lettres en notes via le registre, puis en valeurs numériques.
    *   *Résultat* : Carte Histoire **171** ("L'écho retrouvé").

*   **Énigme 8 : Les Couleurs**
    *   *Objectif* : Associer des couleurs aux cordes.
    *   *Action* : Utiliser une formule mathématique basée sur les couleurs des cordes (Jaune, Vert, Bleu, etc.).
    *   *Résultat* : Carte Histoire **716** ("La chromharmonie").

*   **Énigme 9 : La Projection**
    *   *Objectif* : Projeter l'harmonie.
    *   *Action* : Utiliser le gabarit Lyre sur une "planète colorée" et extraire des numéros selon des filtres.
    *   *Résultat* : Carte Histoire **273** ("Le chant retrouvé").

*   **Énigme 10 : Le Sagittaire**
    *   *Objectif* : Trouver le Sagittaire.
    *   *Action* : Poser le gabarit "Sagittaire" recto sur le ciel et sommer les étoiles.
    *   *Résultat* : Carte Histoire **518** ("L'oubli du sagittaire").

*   **Énigme 11 : Le Faucon**
    *   *Objectif* : Identifier le nom "FAUCON".
    *   *Action* : Utiliser le registre pour épeler FAUCON avec des étoiles et les sommer.
    *   *Résultat* : Carte Histoire **365** ("La rivière qui murmure").

*   **Énigme 12 : Le Labyrinthe**
    *   *Objectif* : Traverser le labyrinthe du Sagittaire.
    *   *Action* : Suivre un chemin sur le verso du gabarit Sagittaire aligné sur le sextant.
    *   *Résultat* : Carte Histoire **380** ("Le port de Zubenelgenubi").

*   **Énigme 13 : La Recette**
    *   *Objectif* : Cuisiner le "Flan de lumière acidulé".
    *   *Action* : Associer ingrédients et gestes pour former le mot SCORPION, puis appliquer une formule.
    *   *Résultat* : Carte Histoire **706** ("Le dessin intérieur").

*   **Énigme 14 : Le Carré de Retissage**
    *   *Objectif* : Révéler une forme cachée.
    *   *Action* : Remplir une grille avec les étoiles roses pour dessiner un cactus.
    *   *Résultat* : Carte Histoire **485** ("Le dernier fragment").

*   **Énigme 15 : Le Final**
    *   *Objectif* : La révélation finale.
    *   *Action* : Reconstituer une phrase avec les dos des cartes histoires ("La ceinture se reconstruit..."), assembler tous les objets physiques et projeter de la lumière à travers des diapositives.
    *   *Résultat* : La projection révèle le message **"Veux tu mepouser"**.
